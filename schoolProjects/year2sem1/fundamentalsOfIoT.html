Fingerprint Door Access

An Arduino project

In this project, me and my teammate created a fingerprint door access system using arduino. Additionally, we created an application to retrieve and send data. The application can act as a controller to unlock/lock the door, send a message through the lcd screen and on/off the arduino. A cloud is used to read and write the data from either the application or the arduino, that is how these two communicates with each other. The cloud platform we used is thingspeak.

The process.

Before starting this project, me and my teammate brainstormed on how we were going to proceed with the project. We ideated, create a concept sketch, block diagram, flowchart, dataflow diagram and a grantt chart. To view the document click here.

After getting the approval of the project from our teacher, we proceeded to gather all the hardware components we needed for this project and set up the thingspeak channel. You can view the thingspeak channel here.

Next, we coded out the arduino program. The program consists of functions, api calls, for loops and if else statements which we learnt in our lectures.

Lastly, we used MIT app inventor to create the application that can be used on android only.
