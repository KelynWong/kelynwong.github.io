Snapsell

An online shop

In this project, I learn to create a simple online shop that interfaces with the database using API endpoints. This website is created from scratch using html, css, javascript nodejs and mysql. The project is split into two parts. The second part being an improvement to the first. To view part 1 codes, click here. To view part 2 codes, click here.

*note: I did not include the node modules so you would have to npm install if you want to try it out on your computer*

Part 1.

Before coding out the API endpoints, I first had to design and create the tables in mysql to prepare for the storage of data. The picture on the left shows the database diagram that I came up with.

I seperated the files from model and controller. The controller consists of all the get, post, put and delete methods and their endpoints. The controller would use the model which consist of sql queries to interface with the database.

Since for this part of the project does not have a front end yet, I used the software called postman to test the APIs out.

Part 2.

For the second part of this project, I have added several improvements like the front end (login page, home page and search page) and additional features to the website.
